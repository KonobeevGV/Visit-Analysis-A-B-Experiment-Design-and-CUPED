# Visit-Analysis-A-B-Experiment-Design-and-CUPED
Анализ поведения пользователей на сайте образовательной платформе: исследование трафика, визуализация динамики и характеристик визитов. Симуляция A/B-экспериментов с проверкой мощности и MDE. Применение CUPED с CatBoost для снижения дисперсии метрик за счёт предэкспериментальных данных.
